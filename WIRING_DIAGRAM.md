# Схема підключення ESP8266 + VL53L0X + 5-сегментний індикатор

## Компоненти:
- ESP8266 (NodeMCU/Wemos D1 Mini)
- VL53L0X лазерний сенсор відстані
- 5-сегментний світлодіодний індикатор рівня (B5R)
- Резистори 220Ω (5 штук для обмеження струму LED)

## Схема підключення:

### VL53L0X → ESP8266:
```
VL53L0X    ESP8266
-------    -------
VCC     →  3.3V
GND     →  GND
SDA     →  D2 (GPIO4)
SCL     →  D1 (GPIO5)
```

### 5-сегментний індикатор → ESP8266:
```
Сегмент    ESP8266    Резистор
--------   -------    --------
Сег.1      D7        220Ω → GND
Сег.2      D6        220Ω → GND  
Сег.3      D5        220Ω → GND
Сег.4      D4        220Ω → GND
Сег.5      D3        220Ω → GND
```

### Живлення:
```
ESP8266:
- VIN/5V → 5V джерело живлення
- GND → Загальний GND

VL53L0X:
- VCC → 3.3V (з ESP8266)
- GND → Загальний GND
```

## Розташування датчика:
1. **Встановіть VL53L0X зверху резервуара/бака**
2. **Направте сенсор вертикально вниз**
3. **Максимальна відстань до дна: 200см**
4. **Мінімальна відстань: 3см**

## Налаштування в коді:
- `MAX_TANK_HEIGHT = 200` - висота вашого бака в см
- `MIN_WATER_LEVEL = 10` - мінімальний рівень спрацювання в см
- `MEASUREMENT_INTERVAL = 2000` - інтервал вимірювань в мілісекундах

## Логіка роботи:
- Рівень води = Висота бака - Виміряна відстань
- Сегменти загораються знизу вгору залежно від рівня води
- При помилці сенсора мигає середній сегмент

## Діагностика:
- Всі сегменти мигають → Помилка ініціалізації VL53L0X
- Середній сегмент мигає → Помилка вимірювання
- Сегменти горять знизу вгору → Нормальна робота
